The settings page allows the user to edit certain display functions, add or remove app features, as well as reset all sessions. The features include a monthly summary, which is added to the Week Select view for a summary of sessions in that month, session times to be added to the View Session Page, as well as if the user would like only unique names in the Main Page to be displayed. By default, the user is in light mode, using the current brightness of the device, and all features are disabled. By clicking on the save icon in the bottom right, the features will be applied (as well as sessions reset confirmed), and the user will return to the Main Page. By clicking on the left arrow in the bottom left, the user will return to the Main Page.

It should be noted that light and dark mode, as well as brightness, are applied automatically rather than on save.
