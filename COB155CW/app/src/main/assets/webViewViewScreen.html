On clicking on any session in the main page, the user is taken to the View Session Page displaying all related information with that session. The user has the ability to delete the task from this view as well as sharing the session information to other downloaded apps. From the View Session page, the user is able to access:

  • Edit Session Screen
    • This can be done by clicking on the pen icon in the bottom right.
    • This will allow the user to update information about the Session.
  • Main Page Screen
    • This can be done by clicking on the back arrow in the bottom left to return.
    • This can also be done on deleting the session by clicking on the bin icon in the top right of the screen.
  • Share Session Display
    • This can be done by clicking on the share icon in the top left.
Note that by default, the session time is disabled so it will have to be enabled in the settings page. It should also be noted that since the above pictures were taken, a new calendar feature was added which when clicked will add the event to the phone's calendar (As seen with the below image).
