The Edit Session Screen is used to create new sessions as well as edit current sessions. This is because the functionality of the tasks is very similar. The main difference between the two operations is when editing the session, it will pre-fill all inputs (based on the current session information) and allow the user to change them. The user uses the text fields, as well as drop-downs and a stopwatch, to fill in all information regarding the session. The value stored in the stopwatch when paused will be the one used (this can be reset back to 0 if inaccurate). On applying the changes to the session (clicking on the tick icon in the bottom right), as well as using the left arrow in the bottom left, the user will be taken back to the Main Page. On creating a new session, the user is sent a notification at 8 am of the day that the session is scheduled, and the notification icon will be of the session type.
