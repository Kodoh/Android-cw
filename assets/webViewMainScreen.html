The main page holds all the current sessions from the given week (which can be found on the top bar), the sessions can be found in a list where the type of session is indicated by a corresponding icon. Within the list, the session name and date can be found. There is also the ability to filter through sessions via the search bar at the top. From the Main Page, the following pages can be accessed:

  • Week Select Screen
    • This can be done by clicking on the date on the top bar.
  • Create Session Screen
    • This can be done by clicking on the + icon in the bottom right of the screen
  • View Session Screen
    • This can be done by clicking on any of the sessions within the list.
    • It will give the information on the session clicked on.
  • Settings Screen
    • This can be done by clicking on the cog icon in the top left of the screen.

It should be noted that by default the date will be the current week but can be changed by going to the "Week Select Screen"
